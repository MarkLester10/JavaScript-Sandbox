Arrow Function

Arrow function is the most important part of JavaScript ES6 module. It makes your code cleaner and shorter.
It uses the Lexical this to persist the `this` keyword when you're accessing it inside a function that points to another instance or make the `this` keyword out of scope

It solves the issue of assigning the `this` keyword to a variable in order to access it inside inner functions

BEFORE:

const someFunction = function(){
const that = this;
function() {
if(that.status){
..// do something
}
  }
}
Arrow Function:

const someFunction = () => {
function() {
if(this.status){
..// do something
}
  }
}
