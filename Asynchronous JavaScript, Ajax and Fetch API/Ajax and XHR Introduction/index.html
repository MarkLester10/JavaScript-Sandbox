Ajax and XHR Introduction

What is Ajax?

  • Asynchronous JavaScript & XML
  • Set of Web Technologies
  • Send and Receive data asynchronously
  • Does not interfere with the current page
  • JSON has replaced XML for the most part
    Most of the api you work with returns JSON Data not XML or some cases both

How it works?

  • Make async requests in the background
  • No page reload / refresh
  • Fetch Data
  • Very Interactive

Ajax allows us to make request asynchronously and it does it behind the scene, without having to reload the entire web page and it is much faster.

XmlHttpRequest (XHR) Object

  • API in the form of an object
  • Provided by the browsers JS Environment
  • Methods transfer data between client/server
  • Can be used with other protocols than HTTP
  • Can work with data other than XML (JSON, plain text)
