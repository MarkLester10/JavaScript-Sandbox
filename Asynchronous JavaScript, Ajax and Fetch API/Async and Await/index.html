Async and Await

Part of the ES7 or ES2016 standard and it's fully implemented in chrome but not sure with the other browser.

Async Function

Adding async to the beginning of a function makes it returns a Promise

async function myFunct() {
return "Hello";
}

myFunct().then((res) => console.log(res));

If you want something to make asynchronous within that function we can use await to actually waits until that's resolved

async function myFunct() {
const promise = new Promise((resolve, reject) => {
setTimeout(() => resolve("Hello World"), 5000);
});

//throw errors
const err = false;
if(!err) {
const res = await promise; //waits until promise is resoved
return res;
} else {
await Promise.reject(new Error("Something went wrong"));
  }
}

myFunct() .then((res) => console.log(res)) .catch((err) => console.log(err));
